    How the weather App works

    this a weather app project that helps you to get to know to the temperatures in the states within USA

    to make this app working on your computer

    1 - you have to install node.js on your computer

    2 - after downloading the project on your pc the open the project by any code editor

    3 - you have to navigate to server.js file

    4 - open your terminal where you are in the server.js file

    installation requirements

    1 - npm install express

    2 - npm install cors

    3 - npm install body-parser

    How to run the weather app

    1 - you need to type in your terminal when you are in the server.js file. 2 - node server.js. 3 - if you installed the requirements correctly then the localhost url or server will appear to you in your terminal. 4 - click on it to open the project at any browser you like it.

    How the weather app works

    1 - type the zip code or the postal code of any state in usa 2 - type your feeling about this day 3 - click on generate botton
